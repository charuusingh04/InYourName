## 📞 Troubleshooting

### Issue: Photos not showing
→ Check `case_01.jpg`, `case_02.jpg`, etc. in `images/photos/` folder
→ File names must be exact (including extension)

### Issue: Music not playing
→ Click the 🎵 button to enable
→ Check browser console for errors
→ Some browsers require user interaction first

### Issue: Animations stuttering on mobile
→ This is rare with modern phones
→ Try closing other apps
→ Clear browser cache
→ Try different browser

### Issue: Hidden page won't unlock
→ Make sure you click the exact heart icon (💌)
→ Must click exactly 7 times
→ Counter resets if you navigate away
→ Try refreshing and clicking again

### Issue: Layout looks wrong
→ Clear browser cache (Ctrl+Shift+Delete)
→ Try different browser
→ Check screen is not zoomed in
→ Use fullscreen for best experience
