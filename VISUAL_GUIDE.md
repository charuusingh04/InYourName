## 🎨 Design Elements Reference

### Color Palette
| Color | Hex | Usage |
|-------|-----|-------|
| White | #FFFFFF | Background |
| Pastel Pink | #FFD6E8 | Accents, borders |
| Dark Pink | #FF99C5 | Headlines, emotional elements |
| Teal | #4A9B9E | Medical theme, buttons |
| Light Teal | #7FB5B7 | Hover states |
| Soft Blue | #B4D7E8 | Placeholders, secondary accents |
| Light Blue | #E0EEF5 | Backgrounds, overlays |
| Light Gray | #F5F5F5 | Subtle backgrounds |

### Animations & Effects
| Animation | Where | Duration |
|-----------|-------|----------|
| ECG Heartbeat | Landing page, Heart monitor | Continuous |
| Confetti Fall | Birthday page | 2-5 seconds |
| Floating Hearts | Forever page | 4 seconds |
| Fade-In | All pages on load | 0.6 seconds |
| Glow | Secret letter, hidden heart | 2 seconds |
| Pulse | Loading animations | 1.5 seconds |
| Slide-in | Card elements | 0.6 seconds (staggered) |
| Heartbeat Vibration | Heart monitor, secret page | Multiple patterns |

### Responsive Breakpoints
| Device | Width | Layout |
|--------|-------|--------|
| Mobile | < 480px | Single column, large text |
| Tablet | 481-768px | Two columns |
| Desktop | > 769px | Multi-column grids |

- 3-second dramatic pause
- Text appears: "This heart belongs to Dr. Charu"

**Location**: Future page

## 💡 Tips for Maximum Impact

1. **Watch Her Face** - Best experienced together

3. **Audio** - Make sure music is enabled
4. **Mobile** - Test on her phone beforehand
5. **Photos** - Use high-quality, meaningful images
6. **Surprise** - Keep the 7-click Easter egg a secret!
